¿Que es una vista en SQL?

Las vistas en MySQL son una forma de simplificar y optimizar el acceso a los datos de una o más tablas. Una vista es una tabla virtual que se crea a partir de una consulta SELECT y que se almacena en la base de datos como un objeto.

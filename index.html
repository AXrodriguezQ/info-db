VIEWS en MYSQL

¿Que es una vista en SQL?

Las vistas en MySQL son una forma de simplificar y optimizar el acceso a los datos de una o más tablas. Una vista es una tabla virtual que se crea a partir de una consulta SELECT y que se almacena en la base de datos como un objeto.

[

OR REPLACE

]

VIEW

nombre_vista [lista_columnas]

AS

consulta;

Procedimientos Almacenados

Un procedimiento almacenado es un conjunto de instrucciones que se pueden llamar desde otras consultas o desde otros procedimientos almacenados. Un procedimiento puede tomar argumentos de entrada y mostrar valores como resultados. Debes nombrar y almacenar un procedimiento en un conjunto de datos de BigQuery.

CREATE PROCEDURE

dataset_name.procedure_name() BEGIN

-- statements here

END

SQL y NoSQL

¿Diferencias entre ellos?

SQL significa Lenguaje de Consulta Estructurado y es un lenguaje para consultar datos estructurados. NoSQL es un sistema de base de datos más nuevo que no utiliza un lenguaje de consulta estándar pero emplea documentos JSON para el almacenamiento de datos.

MONGO DB

Listar en MONGODB

El comando MongoDB List Collections es una forma muy práctica de mostrar todas o algunas colecciones. El comando muestra todos los nombres y opciones de cada una de las listas, pudiendo condicionar el resultado en función de varios parámetros.

Para listar bases de datos en MongoDB puedes utilizar el comando

db.adminCommand

( { listDatabases: 1 } )

Actualizar en MONGODB

Para actualizar documentos en MongoDB, puedes usar los métodos update() y save(). El método update() desplaza los valores en un documento existente, mientras que save() reemplaza el documento existente con el que le pasamos por parámetro.

update()

y

save()

Eliminar en MONGODB

La manera más común de eliminar datos en MongoDB es eliminar documentos individuales de una colección. Puede hacerlo mediante el método deleteOne() o deleteMany(). Es importante tener en cuenta que ambos métodos devuelven un objeto con las siguientes propiedades: deletedCount: la cantidad de documentos eliminados.

deleteOne()

y

deleteMany()

Insercion en MONGODB

Existen fundamentalmente dos métodos para insertar documentos: insertOne() : para insertar los documentos de uno en uno. insertMany() : para insertar varios documentos en un mismo comando.
